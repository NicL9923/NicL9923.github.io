The Motorcyclist's Bible

a React App

Created with: JavaScript, React, Node, Firebase (Backend/API)

Viewable at: themotobible.com

This is by far my top project and continually gets hours thrown into it. In short, it is a motorcyclists' community website featuring a blog, chatroom based in discord, minigames centered on riding knowledge and skills, and other various resources related to motorcycles. This is also where I delved deepest into React and now have a very good familiarity with the library. Initially this project was meant to have its own custom backend in MongoDB, express, and Node but due to needing a quick and free hosting solution, Firebase caught on and has been in use since. It still involves some backend work (database, queries, API calls, etc.), but obviously at a slightly higher leve of abstraction so take that as you will.
